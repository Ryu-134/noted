## NOTED (Student Notes OS)

Phase 1 scaffold for the browser-first note transformation project documented in `NOTED_Project_Doc_1-8-26`.

### Phase 1 Status

- Completed: repository setup
- Completed: Next.js App Router scaffold
- Completed: TypeScript + Tailwind configuration
- Completed: starter shell page
- Ready: Vercel deployment

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build + Production Run

```bash
npm run build
npm run start
```

### Vercel Deployment

1. Push this repository to GitHub.
2. Import the repo into Vercel.
3. Keep defaults for a standard Next.js project.
4. Deploy to get a public Phase 1 starter shell.

### Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- ESLint
